# 🚀 SCARLA Quiz API Backend

Backend API untuk SCARLA Learning Platform yang menyediakan semua materi pembelajaran dan quiz.

## 📚 Materi yang Tersedia

- **HTML dasar** - 3 video
- **CSS dasar** - 3 video
- **JavaScript dasar** - 4 video
- **Java dasar** - 4 video
- **Python dasar** - 4 video
- **C# dasar** - 4 video

Setiap materi dilengkapi dengan **5 soal quiz**.
